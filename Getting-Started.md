# Getting Started
 
To get started using Actum's API, you'll want to follow these steps:
 
### Step 1: Register for a Test Account
 
First, you will want to navigate to Actum's [Register for a Test Account](https://www.actumprocessing.com/register-test-account/) page. You will need to enter your First Name, Last Name, Email, Company Name, Phone Number, and, optionally, specify if you are a current partner of Actum. Actum's support team will use this information for verification purposes.
